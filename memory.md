# 博客项目记忆库

这是本项目相关的开发记忆与规则记录，用于指导后续的开发与维护工作。

## 核心规则与偏好

- **代码修改边界限制**：当被要求修改指定页面或添加某个特定功能时，绝不擅自修改其他页面的内容或代码，保持严格的作用域控制。
- **记录踩坑经验**：今后在博客项目中遇到错误修复或踩坑经验时，必须将这些错误主动更新记录到项目的 `readme.md` 文件中。
- **记忆与技能存储规范**：当被要求“创建记忆”时，必须将该记忆的文本内容同步写入到本项目的 `memory.md` 文件中；当被要求“创建 skill”时，必须直接创建并写入到一个名为 `skill.md` 的文件中。
