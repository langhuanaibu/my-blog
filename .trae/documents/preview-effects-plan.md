# 博客视觉效果调整计划

## 摘要

1. **首页雪花粒子（浅色模式）**：颜色太淡，与浅色背景对比度不够 → 调深颜色让雪花更显眼
2. **文章页格子纹理（浅色+深色）**：太淡 → 调大透明度
3. 创建 `source/preview-effects.html` 预览页面，实时调节参数后确定最终值

---

## 当前状态分析

### 雪花粒子 (`source/js/aoiblog-home.js` 第 22-110 行)

- Canvas 2D 实现，仅首页触发（`isHome` 为 true）
- 三层粒子系统：90 + 120 + 42 = 252 个雪花
- **浅色模式颜色**（第 48 行）：`rgb(255,248,235)`、`rgb(236,198,154)`、`rgb(208,145,88)` — 暖米色系，与背景 `#ead8c2` 太接近，几乎看不见
- **深色模式颜色**（第 47 行）：`rgb(255,226,190)`、`rgb(242,199,159)`、`rgb(224,173,120)` — 与深色背景对比足够
- 各层参数（第 70-72 行）：第1层 90颗 1-2.4px 慢 / 第2层 120颗 2.4-4.8px 中 / 第3层 42颗 4.8-8.2px 快

### 格子纹理 (`source/css/aoiblog-home.css` 第 161-183 行)

- JS `addPaperTexture()` 动态创建 `<div class="aoiblog-paper-texture">`
- 两个 `repeating-linear-gradient` 叠加：间距 23px，线宽 1px
- **浅色模式**：`opacity: 0.08`，线条 `rgba(111,61,32,0.7)` / `rgba(111,61,32,0.55)`
- **深色模式**：`opacity: 0.07`，线条 `rgba(224,173,120,0.55)` / `rgba(224,173,120,0.42)`
- `mask-image` 使左右边缘渐隐
- 仅文章页触发（`isPost` 为 true）

## 方案设计

### 第一步：创建预览页面 `source/preview-effects.html`

独立的预览调试页面，包含：

1. **双模式切换**：浅色模式（羊皮纸背景 `#ead8c2`）/ 深色模式（暗色背景 `#1d1712`），切换时同步更新两个预览区
2. **雪花预览区**（模拟首页）：Canvas 雪花粒子，带滑块控件：
   - 浅色模式：三色 RGB、透明度范围
   - 深色模式：三色 RGB、透明度范围
   - 通用：粒子数量、速度、大小
3. **格子纹理预览区**（模拟文章页）：`repeating-linear-gradient` 格子，带滑块控件：
   - 浅色/深色各自独立的：整体 `opacity`、线条颜色 RGB/alpha、网格间距、线宽
4. 显示当前参数值，方便确认后直接写入源码

### 第二步：用户调节确认参数

打开预览页面，切换浅色/深色模式分别调节参数，确定满意的值。

### 第三步：修改源码

#### `source/js/aoiblog-home.js`

- 第 48 行：根据预览确认的浅色模式雪花颜色值，替换 `colors` 数组

#### `source/css/aoiblog-home.css`

- 第 168 行：浅色模式 `opacity` 调大
- 第 179 行：深色模式 `opacity` 调大
- 如预览需要，同步调整线条颜色
