# 个人博客 · 每日新闻日报

个人自用的每日新闻日报：从多信源抓取、打分筛选，生成一份供桌面端每天一次通读的简报。它是**个人化信号筛选器**，不是新闻流。

本文件只是术语表，用于统一人与 agent 的用词。实现细节见 `readme.md` 与 `docs/`。

## Language

### 日报结构

**日报**：
某一天产出的那一份简报，含主线、精选与各附属频道。
_Avoid_：新闻流、资讯流

**精选**：
当天通过分数线、进入正文逐条通读的条目。
_Avoid_：头条、推荐

**次级**：
未进精选、只以标题形式列在「更多资讯」里的条目。
_Avoid_：备选、低分条目

**今日主线**：
把当天若干相关精选归拢成的一条当日综合线索。仅描述当天，不跨天。
_Avoid_：主题、话题、热点

**事件线**：
一件持续发展的具体事情跨天形成的链条，是跨天追踪的最小单位。同一主体的不同事情不算同一条事件线。
_Avoid_：主题、话题、专题

### 条目内容

一条精选按 **来龙 → 现状 → 走向** 的顺序讲述，另附「对我的意义」。

**来龙**：
这条事件线怎样一步步走到今天。只由已验证的历史构成，不是泛泛的领域背景。
_Avoid_：背景、背景机制、前情提要

**现状**：
今天发生了什么、以及它影响谁、为何重要。
_Avoid_：摘要、正文

**走向**：
接下来取决于什么、以及用什么可观察的变化来判断它往哪走。
_Avoid_：后续关注、预测、前瞻

**对我的意义**：
这条新闻对读者本人的学习或行动参考。
_Avoid_：启示、行动项

### 走向的构成

**关键变量**：
接下来决定这条事件线走向的少数几个因素。
_Avoid_：影响因素、驱动力

**可观察路标**：
一个可证伪的具体信号——看到它就能判断走向。没有路标的走向不成立。
_Avoid_：预测、指标、信号灯

**参考类比**：
历史上可比的同类情形及其收场方式，用来给走向定锚。只在材料明确支持时使用。
_Avoid_：先例、类比推理

### 延续与信任

**可信延续**：
今天这条新闻确实是某条既有事件线的后续，且该判断已通过验证。**仅仅被匹配到同一登记事件不算可信延续。**
_Avoid_：延续、追踪中、day_count≥2

**连续性门**：
判定「今天这条是否真属于那条事件线」的准入检查。不通过就按新事件处理。
_Avoid_：去重、相似度匹配

**走向回对**：
在后续报道中说明上一期的走向路标究竟兑现、部分兑现、未兑现还是反转。只在证据足够时给出结论。
_Avoid_：复盘、打分、准确率
