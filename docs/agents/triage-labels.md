# Triage Labels

工程 Skills 使用以下五个标准角色。右列是本仓库 GitHub Issues 中对应的实际标签。

| Skill role | GitHub label | Meaning |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 待维护者评估 |
| `needs-info` | `needs-info` | 等待报告者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 需求已定，可由 agent 实现 |
| `ready-for-human` | `ready-for-human` | 需要人工处理或决策 |
| `wontfix` | `wontfix` | 已决定不处理 |

当 Skill 提到某个 triage 角色时，使用表中对应的 GitHub 标签。
