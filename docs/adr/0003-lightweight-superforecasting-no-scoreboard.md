# 走向采用轻量超预测法，且不建预测记分板

「分析走势与可能结果」若无方法约束就会滑向占卜。我们借用超预测法（Tetlock / Good Judgment Project）中最轻的一层——**关键变量 + 可观察路标**，参考类比仅在材料明确支持时使用——并**禁止具体概率数字、无条件断言和来源外类比**。独立的分析结论进入 `claims[]` 受审计；条件式的观察路标由轨迹审计直接检查，不在 `claims[]` 里重复一遍。

超预测法的核心纪律本是 keep score（记录预测、回填对错、追准确率），但本项目此前已验证：行动状态机、周复盘一类仪式的实际使用率近零。因此**不建预测记分板**。问责改由两点承担：每条走向必须写成可证伪的路标；后续报道在证据足够时做**走向回对**（兑现／部分兑现／未兑现／反转）。

## Consequences

日报侧的走向回对沿用周报 `watch_recap` 已有的状态枚举与证据绑定范式，而不是另造一套措辞。这意味着日报需要能取到上一期最终展示的 `watch`。
